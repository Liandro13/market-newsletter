## 📋 About

**Market Newsletter** is a fully automated pipeline that runs daily via GitHub Actions. It collects financial market data (stocks, indices, crypto) using `yfinance`, analyzes it with a large language model via the Groq API, and delivers the result as an email through Resend — all for free.

## ⚙️ How It Works

```
⏰ GitHub Actions (daily schedule)
    ↓
📊 yfinance — fetch market data (prices, changes, volumes)
    ↓
🤖 Groq API (LLM) — AI-powered analysis and newsletter writing
    ↓
📧 Resend — deliver to inbox
```

## ✨ Features

- **Fully automated** — runs daily with no manual intervention
- **Real-time data** — live quotes and market movements via yfinance
- **AI-written analysis** — narrative generated by an LLM (Groq)
- **Email delivery** — straight to your inbox via Resend
- **100% free** — all tools used on their free tier

## 🛠️ Tech Stack

| Tool | Role |
|---|---|
| `yfinance` | Financial data (Yahoo Finance) |
| `Groq API` | High-speed LLM inference |
| `Resend` | Transactional email delivery |
| `GitHub Actions` | Scheduling and automation |

## 🔑 Configuration

Add the following **Secrets** to the repository (`Settings → Secrets → Actions`):

| Secret | Description |
|---|---|
| `GROQ_API_KEY` | Your Groq API key |
| `RESEND_API_KEY` | Your Resend API key |
| `EMAIL_TO` | Destination email address |

## 🚀 Running Locally

```bash
git clone https://github.com/Liandro13/market-newsletter.git
cd market-newsletter

pip install yfinance groq resend

export GROQ_API_KEY="..."
export RESEND_API_KEY="..."
export EMAIL_TO="..."

python main.py
```

## ⏰ Schedule

The GitHub Actions workflow runs automatically on weekdays. Adjust the cron expression in `.github/workflows/newsletter.yml` to change the schedule.
